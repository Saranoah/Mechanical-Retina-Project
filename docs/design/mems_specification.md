# MEMS Design Specifications

## Cantilever Array Design

### Pixel Specifications
- Array size: 100 × 100 pixels
- Pixel pitch: 50 μm
- Cantilever dimensions: 40 × 10 × 0.5 μm
- Material: Single crystal silicon

### Performance Targets
- Spring constant: 0.1 N/m
- Resonance frequency: 100 kHz
- Quality factor: > 10,000 (vacuum)
- Minimum detectable force: < 1 fN/√Hz

### Readout Methods
1. **Optical**: Laser interferometry
2. **Capacitive**: Interdigitated electrodes
3. **Piezoresistive**: Doped silicon strain gauges

## Fabrication Process
1. SOI wafer preparation
2. Lithographic patterning
3. DRIE etching
4. Sacrificial layer release
5. Metallization
6. Packaging
