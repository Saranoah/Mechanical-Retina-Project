# Physics Background

## Fundamental Equations

### Photon Properties
- Energy: E = hc/λ
- Momentum: p = h/λ
- Radiation pressure: P_rad = I/c (absorption), 2I/c (reflection)

### Mechanical Response
- Cantilever spring constant: k = Ew³t/4L³
- Thermal noise: x_rms = √(kBT/k)
- Minimum detectable force: F_min = √(4kBTB/kω₀Q)

### Signal-to-Noise Considerations
- Standard Quantum Limit for displacement detection
- Thermal Brownian motion limitations
- Photon shot noise vs. thermal mechanical noise
