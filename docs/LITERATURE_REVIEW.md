# Literature Review

## Key Papers

### Radiation Pressure Detection
1. **"Laser Cooling and Control of Micromechanical Oscillators"** (2011)
   - Demonstrates radiation pressure effects on micro-mirrors
   - Requires cryogenic temperatures for single-photon sensitivity

2. **"Optomechanical Photon Counting"** (2018)  
   - Theoretical framework for photon counting via mechanical displacement
   - Highlights thermal noise challenges

### MEMS Photon Sensors
3. **"Nanomechanical Bolometers"** (2020)
   - Thermal detection using nanomechanical resonators
   - Achieves NEP of 10⁻¹⁷ W/√Hz at room temperature

### Photomechanical Materials
4. **"Light-Induced Shape Changes in Azobenzene Polymers"** (2019)
   - Documents mechanical response to photon absorption
   - Potential for direct photon-to-mechanical transduction

## Technology Gaps
- Room temperature single-photon mechanical detection
- Scalable array fabrication
- Efficient readout of mechanical displacements
- Real-time image reconstruction from mechanical signals
