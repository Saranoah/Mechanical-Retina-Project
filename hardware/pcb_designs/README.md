# PCB Designs for Mechanical Retina

## Readout Board
- **Purpose**: Interface with mechanical sensor array
- **Key Components**:
  - Low-noise amplifiers
  - High-resolution ADC (24-bit)
  - FPGA for real-time processing
  - USB 3.0 interface

## Power Supply Board
- **Purpose**: Provide clean power to sensitive analog circuits
- **Features**:
  - Low-noise linear regulators
  - Active filtering
  - Over-voltage protection

## Control Board
- **Purpose**: System management and thermal control
- **Functions**:
  - Temperature monitoring
  - Piezoelectric actuator drivers
  - System status indicators
